# Go API client for edgework_sdk

Unified AI Gateway API for accessing multiple LLM providers through a single interface.

## Overview
The AI Gateway provides a unified interface to Affectively's self-hosted distributed inference running on Cloudflare Workers and Google Cloud Run.
All models are open-source and self-hosted - no paid third-party AI providers are used.

- **Vision/Image Generation**: flux-4b

## Token Counting
Tokens are the fundamental unit of rate limiting in the AI Gateway.
All models are self-hosted with no per-token costs.

### What are Tokens?
- **Input tokens**: Text you send to the AI model (typically ~4 characters per token)
- **Output tokens**: Text the AI generates (variable, typically longer than input)

### Token Examples
- \"Hello\" ≈ 1 token
- A paragraph ≈ 50-100 tokens
- A page ≈ 300-500 tokens
- An image (via vision models) ≈ 85-130 tokens

### Optimization Tips
- Use shorter prompts and be specific about requirements
- For ensemble models, input tokens are counted once but output scales with model count
- Streaming responses count the same as non-streaming
