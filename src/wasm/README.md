# WASM Loader

The **WASM** module handles the loading and initialization of WebAssembly binaries for the SDK.

## Features

- **Loader (`loader.ts`)**: Robust loading strategy that handles browser vs Node.js environments.
- **Caching**: Efficient caching of compiled WASM modules to prevent re-instantiation costs.

## Architecture

Essential for high-performance crypto operations and local AI inference running in the browser.
